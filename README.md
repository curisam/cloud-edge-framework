# cloud-edge-framework

- 클라우드와 에지가 함께 기계학습 처리를 수행하는 프레임워크 기술입니다.
- 분산된 환경에서 협력학습을 통해 성능은 높이고, 개인정보 이슈는 극복합니다.
- 종래의 연합학습 프레임워크 기술을 적극적으로 수용하면서, 부족한 부분은 채워줍니다.
- 서버급 고성능 컴퓨터에서부터 라즈베리파이와 같은 저사양 컴퓨터까지 함께 사용할 수 있습니다.
